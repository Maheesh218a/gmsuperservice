# GM Super Service

![GM Super Service Logo](https://i.ibb.co/nqNX7GjJ/LOGO.png)

Welcome to the **GM Super Service** website repository! This is a modern, luxury transportation service platform built to provide an elevated travel experience in Sri Lanka. 

## 🌐 Live Website
**Visit the live site here:** [https://gmsuperservice.vercel.app/](https://gmsuperservice.vercel.app/)

---

## ✨ Features
- **Premium Design Aesthetics**: Smooth glassmorphism UI, elegant gold accents, and stunning animations powered by Framer Motion.
- **Dynamic Fleet Display**: Showcase of luxury buses, cars, and vans with dedicated detail pages including dynamic feature lists, seat mapping capabilities, and image galleries.
- **Interactive Image Lightbox**: Fully custom responsive image galleries with exact-file download functionality and direct Google Drive integrations.
- **Booking System**: Users can submit travel bookings directly from the Contact page which are securely saved to an administrative database.
- **Secure Admin Dashboard**: Features an Admin Panel shielded by Firebase Authentication and custom Google email-driven Two-Factor Authentication (2FA).
- **Admin Image Management**: Full Create/Read/Update/Delete (CRUD) capabilities via Firebase to easily manage website images directly from the browser.

## 🛠 Tech Stack
- **Frontend Framework:** React (using Vite)
- **Styling:** Tailwind CSS + Custom CSS properties
- **Animations:** Framer Motion
- **Database & Auth:** Google Firebase (Firestore, Authentication)
- **Icons:** React Icons
- **Deployment:** Vercel

## 🚀 Getting Started Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Maheesh218a/gmsuperservice.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd gmsuperservice
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Run the development server:**
   ```bash
   npm run dev
   ```
